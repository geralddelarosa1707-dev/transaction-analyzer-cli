# Transaction Analyzer (CLI App)

## Description
A Python CLI application for managing, analyzing, and storing sales and expense transactions using JSON.

## Features
- Add sales and expenses
- View current transactions
- View past transactions
- Save transactions by date
- Search and Update items
- Search Transaction
- Remove items
- Delete transactions
- Calculate profit/loss

## Technologies Used
- Python
- JSON

## How to Run
1. Run the script:
   python main.py
2. Follow the menu options

## Notes
- Data is stored locally using a JSON file (`transactions.json`)
- Make sure Python is installed on your system
